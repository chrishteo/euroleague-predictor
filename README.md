# EuroLeague Predictor

A Monte Carlo simulation-based prediction system for the EuroLeague basketball season. Predict final standings, playoff probabilities, and track team performance throughout the season.

## ğŸ“Š Prediction Methodology

### Team Rating Calculation

Each team receives an ELO-style rating:

```
Rating = 1500 + (WinPct - 0.5) Ã— 400 + AvgMargin Ã— 10
```

Where:
- `WinPct` = Wins / Games Played
- `AvgMargin` = (Points For - Points Against) / Games Played

### Win Probability

For each simulated game:

```
P(Win) = 1 / (1 + 10^((OpponentRating - TeamRating - HomeAdvantage) / 400))
```

Home advantage is set to +50 rating points (~3-4 point spread).

### Simulation Process

1. Calculate current ratings for all teams
2. For each of 5,000 simulations:
   - Simulate remaining games for each team
   - Randomly assign opponents (simplified scheduling)
   - Determine winner based on win probability
   - Record final standings
3. Aggregate results to calculate position probabilities

## âš ï¸ Limitations

1. **CORS Restrictions**: The EuroLeague API cannot be called directly from browsers due to CORS policies. Use the Python script for data fetching.

2. **Simplified Scheduling**: The simulation uses random opponent assignment rather than the actual remaining schedule.

3. **No Player-Level Modeling**: Current predictions are team-based only. Player injuries, transfers, and individual performance are not factored in.

4. **Head-to-Head Tiebreakers**: The app uses point differential as the tiebreaker, not the official EuroLeague head-to-head rules.
